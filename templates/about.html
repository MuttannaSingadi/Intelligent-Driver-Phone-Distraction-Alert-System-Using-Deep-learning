Driver Phone Distraction Alert system

"Welcome to Driver Phone Distraction Alert system" is an intelligent driver monitoring system designed to enhance road safety by detecting and classifying distracted driving behaviors in real time. Using deep learning and computer vision, the system identifies 10 types of driver actions, including safe driving, texting, phone usage, drinking, and other distractions.

What are we trying to do?

In this project, we aim to develop an AI-powered system that can automatically detect and classify driver behaviors to identify whether the driver is attentive or distracted. By analyzing real-time video input from a camera inside the vehicle, the system recognizes specific actions such as texting, calling, drinking, or safely driving.

Data Collection

Gather a dataset of images labeled with their corresponding categories

Data Preprocessing

Resize the images to a consistent size, normalize pixel values, and augment the dataset if necessary to increase the diversity of training samples.

Model Architecture Design

Design a CNN architecture suitable for image classification tasks. This may involve stacking convolutional layers, pooling layers, and fully connected layers, possibly with dropout regularization to prevent overfitting.

Model Training

Split the dataset into training, validation, and test sets. Train the CNN model on the training set, using the validation set to tune hyperparameters and monitor model performance.

Model Evaluation

Evaluate the trained model's performance on the test set using metrics such as accuracy, precision, recall, and F1-score.

Model Deployment

Deploy the trained model as part of an application or service where it can classify Deep Fake images in real-time or in batch processing mode.
